My Portfolio

Sentiment Analysis

Created a logistic regression model to analyze product reviews. Used GridSearchCV for tuning hyperparameters. Achieved an accuracy of 90% while classifying a review as positive or negative.

Smart irrigation and plant disease detection

Implemented an IoT system that automatically release water to the fields depending on factors like temperature, soil moisture, and humidity using Arduino and Think Cloud. Created a website to display these values as well as a Plant Disease Detection system using a CNN-based real-time classifier with train and test accuracy of 99 per cent and 96 per cent respectively. Published two research papers.

Smart Parking

Developed a working prototype for smart parking system that grants access only to authorized personnel. An LED lights up according to the availability of a parking space. Presented in the mini-project competition in the department and won third place.
